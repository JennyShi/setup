ReadMe:


1.	Please put  “rally_api_emc_sso-1.0.0.gem”  and the batch file “setup.bat” under c:\download”

2.	Click on the batch file, it will request you to download Github for windows.

3.	Please install it into default directory like C:\Program Files\Git.

4.	After installing the app, press any button to continue.

5.	It will create a folder under  user\username  named rally Setup, 

	and copy all pfs-feature files and sso gem into it.

6.	If gem exists, it will install. Else, it will give you some prompt like can’t find the file and exit.
